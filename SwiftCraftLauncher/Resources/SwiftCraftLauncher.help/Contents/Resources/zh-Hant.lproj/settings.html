啟動器設定

Swift Craft Launcher 提供了多種設定選項，讓您自訂使用體驗。

一般設定

語言

您可以設定啟動器介面語言。變更將在重新啟動啟動器後生效。

外觀

您可以將啟動器外觀設定為淺色、深色或跟隨系統。效果如下所示。

工作目錄

您可以設定啟動器的工作目錄。啟動器管理的所有遊戲檔案都將放在此資料夾中，如下所示。

重要：變更工作目錄後，原始路徑中的遊戲將不再出現在「遊戲清單」中。建議轉移或重新下載它們。最好在首次開啟啟動器時變更工作目錄。

下載設定

並行下載數量在很大程度上決定了您的下載速度，但更高的並發性也會帶來更大的效能開銷。

如果您不熟悉 Minecraft 版本 URL、Modrinth API URL 和 Git 代理 URL 的含義，請不要修改它們。變更它們可能導致遊戲資源下載失敗。

如果您意外變更了它們，請將它們恢復為預設值：

  • Minecraft 版本 URL：https://launchermeta.mojang.com/mc/game/version_manifest.json
  • Modrinth API URL：https://api.modrinth.com/v2
  • Git 代理 URL：https://ghfast.top

標籤頁

資料包功能正在開發中。請關注更新。

遊戲設定

自動依賴處理

啟用此選項後，下載資源所需的任何依賴項將自動下載，不會彈出通知。以下頁面將不會出現。

Java 路徑

您可以變更此選項以選擇不同的 Java 版本來啟動遊戲。

以下是 MC 版本及其最低要求的 Java 版本：

  • 1.12 - Java 8
  • 1.17 - Java 16
  • 1.18 - Java 17
  • 1.20.5 - Java 21 (64-bit)

全域記憶體分配

您可以控制分配給遊戲的記憶體量。最小值為 512MB。通常，較大的模組包需要更多記憶體。
