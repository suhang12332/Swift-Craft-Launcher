新增資源

Swift Craft Launcher 支援從 CurseForge 和 Modrinth 自動下載模組、資源包（或紋理包）、著色器和資料包，並將它們新增到遊戲中。

新增模組

重要：Vanilla Minecraft 不支援模組。如果您想安裝模組，請確保您已正確安裝了模組載入器。

  1. 在左側的「遊戲清單」中，選擇您要為其安裝模組的遊戲，然後點選頂部按鈕進入「資源庫」。
  2. 選擇頂部的「模組」選項，使用右側的搜尋欄查找您要新增的模組。找到模組後，點選「安裝」。
  3. 某些模組可能會顯示如圖所示的介面。這是因為模組需要其他模組作為依賴項才能執行。不用擔心，點選「下載所有依賴項並繼續」後，啟動器會自動為您安裝它們。模組安裝完成後，點選頂部按鈕返回「已安裝」。

    如果您不想使用自動依賴項安裝功能，可以在啟動器設定中將其關閉。

  4. 當您剛剛下載的模組出現在「已安裝」頁面上時，表示模組已安裝。
  5. 此時，啟動遊戲將顯示模組已成功載入。

新增資源包

  1. 在左側的「遊戲清單」中，選擇您要為其安裝資源包的遊戲，然後點選頂部按鈕進入「資源庫」。
  2. 選擇頂部的「資源包」選項，使用右側的搜尋欄查找您要新增的資源包。找到合適的資源包後，點選「安裝」。完成後，點選頂部按鈕返回「已安裝」。
  3. 當「已安裝」頁面顯示您剛剛下載的資源包時，表示資源包已安裝。
  4. 啟動遊戲，您將看到資源包已被成功識別。點選小三角形以載入資源包。

新增著色器

前提條件：如果您想安裝著色器，請確保您已正確安裝了以下之一：Optifine、Iris Shader 或 Oculus。

  1. 在左側的「遊戲清單」中，選擇您要為其安裝著色器的遊戲，然後點選頂部按鈕進入「資源庫」。
  2. 選擇頂部的「著色器」選項，使用右側的搜尋欄查找您要新增的著色器。找到著色器後，點選「安裝」。完成後，點選頂部按鈕返回「已安裝」。
  3. 當「已安裝」頁面顯示您剛剛下載的著色器時，表示著色器已安裝。
  4. 啟動遊戲，您將看到著色器已被成功識別。點選您下載的著色器以載入著色器包。

新增資料包

資料包為玩家提供了更多自訂 Minecraft 遊戲內容的方式，包括但不限於進度、配方、戰利品表、附魔、傷害類型、生物變體和世界生成等。

注意：新增資料包功能仍在開發中。請關注更新。

手動新增資源

如果您在資源庫中找不到所需的資源，可以手動將資源新增到遊戲中。

請先準備您的資源檔案：模組 (.jar)、資源包 (.zip)、資料包 (.zip)、著色器包 (.zip)、設定檔 (.toml, .json, .yaml 等) 和存檔檔案（資料夾）。

  1. 在左側的「遊戲清單」中，選擇您要新增資源的遊戲，然後點選頂部的「路徑」按鈕。
  2. 啟動器將開啟「Finder」並自動跳轉到遊戲的目錄，如下所示。
  3. 將資源檔案放入相應的資料夾：
    • config：設定檔
    • mods：模組檔案
    • resourcespack：資源包檔案
    • saves：存檔檔案
    • shaderpacks：著色器包檔案

如果您新增設定檔或模組，需要重新啟動遊戲才能生效；其他資源無需重新啟動。

新增模組包

  1. 在左側的「資源清單」中，選擇「模組包」。在頂部搜尋欄中，搜尋您要新增的模組包，然後點選「安裝」。
  2. 選擇「遊戲版本」和「模組包版本」，命名版本，然後點選「下載」。
  3. 啟動器將自動為您安裝 Java 和模組包。
  4. 已安裝的模組包將出現在左側的「遊戲清單」中。
