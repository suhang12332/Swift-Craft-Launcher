Swift Craft Launcher Help

Welcome to Swift Craft Launcher, a native macOS Minecraft launcher built with SwiftUI. This help guide will walk you through all the features and how to use them.

Note: This document is based on version 0.2.6-beta(26). The launcher and documentation are still in the early stages of development, so there may be some inconsistencies. We appreciate your understanding.
