启动器设置

Swift Craft Launcher 提供了多种配置选项，让您自定义使用体验。

常规设置

语言

您可以设置启动器界面语言。更改将在重启启动器后生效。

外观

您可以将启动器外观设置为浅色、深色或跟随系统。效果如下所示。

工作目录

您可以设置启动器的工作目录。启动器管理的所有游戏文件都将放在此文件夹中，如下所示。

重要：更改工作目录后，原始路径中的游戏将不再出现在"游戏列表"中。建议转移或重新下载它们。最好在首次打开启动器时更改工作目录。

下载设置

并行下载数量在很大程度上决定了您的下载速度，但更高的并发性也会带来更大的性能开销。

如果您不熟悉 Minecraft 版本 URL、Modrinth API URL 和 Git 代理 URL 的含义，请不要修改它们。更改它们可能导致游戏资源下载失败。

如果您意外更改了它们，请将它们恢复为默认值：

  • Minecraft 版本 URL：https://launchermeta.mojang.com/mc/game/version_manifest.json
  • Modrinth API URL：https://api.modrinth.com/v2
  • Git 代理 URL：https://ghfast.top

游戏设置

自动依赖处理

启用此选项后，下载资源所需的任何依赖项将自动下载，不会弹出通知。以下页面将不会出现。

Java 路径

您可以更改此选项以选择不同的 Java 版本来启动游戏。

以下是 MC 版本及其最低要求的 Java 版本：

  • 1.12 - Java 8
  • 1.17 - Java 16
  • 1.18 - Java 17
  • 1.20.5 - Java 21 (64-bit)

全局内存分配

您可以控制分配给游戏的内存量。最小值为 512MB。通常，较大的模组包需要更多内存。
