# 安全策略

*[English](../SECURITY.md) | [简体中文](SECURITY.md) | 繁體中文

感謝你關注 **Swift Craft Launcher** 的安全！  
本專案雖然主要面向開發與學習，但我們仍然重視安全問題。

---

## 支援的版本

我們目前僅對以下分支提供安全更新：

- dev
- main

---

## 如何報告漏洞

**請不要在公共 Issue 中直接報告安全漏洞！**  
這樣可能會讓其他使用者暴露在風險之中。

如果你發現了漏洞，請透過以下方式私下聯繫：

- 最推薦：進入我們的 QQ 群跟群主回報，或使用 Swift Craft Launcher 官方信箱：`swiftcraftlauncher@outlook.com`
- 或使用 GitHub 的 **Security Advisories** 功能提交報告
- 提交給倉庫貢獻者

---

## 報告內容建議

為了幫助我們快速修復，請盡可能提供：

1. **漏洞描述**：問題是什麼，潛在影響是什麼
2. **重現步驟**：怎樣觸發漏洞（如果能穩定重現更好）
3. **執行環境**：作業系統版本、Swift Craft Launcher 版本
4. **紀錄/截圖**：有助於理解問題的相關資訊

---

## 我們的承諾

- 我們會盡快確認收到你的報告
- 會在合理的時間範圍內修復漏洞
- 修復完成後，會發布更新並在變更日誌中說明
- 在沒有修復前，我們不會公開披露細節

---

## 致謝

安全依賴於每一位使用者的幫助！  
感謝你為 Swift Craft Launcher 的安全做出的貢獻。
